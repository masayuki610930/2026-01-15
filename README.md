# ビンゴアプリ（参加者カード + 司会者抽選）

シンプルなHTML/CSS/JavaScriptだけで動くビンゴツールです。参加者は手元のカードをタップし、司会者は伏せカードをめくって番号を進行します。GitHub Pagesでの公開にそのまま使えます。

## 画面と仕様
- 参加者画面 `index.html`
  - 5x5カードを自動生成（中央FREE）。列ごとの数字範囲は公式ルール準拠: B 1-15 / I 16-30 / N 31-45 / G 46-60 / O 61-75。
  - マスをタップでチェック。リーチ中の欠けマスを点滅表示。ビンゴ成立時は2秒間の「BINGO!」オーバーレイを表示（クリックでも即閉じ）。
  - リーチ/ビンゴ数のカウントと盛り上げメッセージを表示。
- 司会者画面 `host.html`
  - 1〜75をシャッフルしたデッキから5枚の伏せカードに事前配布。クリックで割り当て済みの番号をめくり、次の番号を自動で再配布。
  - 長押しでそのカードの割当番号を任意指定（他カードと重複しないよう調整、既出番号は不可）。
  - ランダムで1枚めくるボタン付き。出た順と番号順の履歴を表示。
  - 右上に参加者画面へのQRコード（`qr-participants.png`）をリンク付きで配置。

## 動かし方（ローカル）
```bash
cd /path/to/2026-01-15
python -m http.server 8000   # または npx serve .
```
- 参加者画面: http://localhost:8000/
- 司会者画面: http://localhost:8000/host.html

## GitHub Pages で公開
1. リポジトリを GitHub にプッシュ（例: `main` ブランチ）。
2. リポジトリ Settings > Pages で Branch を `main` / `/ (root)` に設定。
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開。

## カスタマイズ
- 参加者カード生成や演出: `app.js` / スタイル: `styles.css`
- 司会者ロジックやUI: `host.js`, `host.html`
- QR画像を差し替える場合は `qr-participants.png` を置き換え、`host.html` のパスを変更してください。
