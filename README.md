# ビンゴアプリ

司会者が番号を引き、参加者が手元で盤面をチェックできるシンプルなビンゴツールです。HTML/CSS/JavaScriptのみで構成されているため、ローカル実行とGitHub Pagesでの公開が容易です。

## 機能
- `index.html`: 参加者用カード。リーチ/ビンゴまでの盛り上げメッセージ付き。
- `host.html`: 司会者（ゲームマスター）用の番号抽選と履歴一覧。
- 1〜75の番号をシャッフルし、司会者が1つずつ引ける（履歴は最新を強調）
- 5x5のビンゴカード自動生成（中央FREE、ライン数とリーチ数を自動カウント）
- マウス/タップでマスをチェック、モバイルでも利用可能

## ローカルでの動かし方
ブラウザで開くだけでも動きますが、ローカルサーバーを立てると安心です。
```bash
cd /path/to/2026-01-15
python -m http.server 8000
# または: npx serve .
```
起動後、ブラウザで `http://localhost:8000` を開きます。

司会者ページは `http://localhost:8000/host.html` で表示できます。

## GitHub Pagesへのデプロイ
1. このフォルダ一式をGitHubリポジトリにプッシュ（例: `main` ブランチ）。
2. GitHubのリポジトリ設定で **Pages** を開き、Branchを `main` / `/ (root)` に設定して保存。
3. 数分待つと `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

## カスタマイズのヒント
- `TOTAL_NUMBERS` を変更すると出す番号の上限を変えられます（`app.js`）。
- スタイル調整は `styles.css` を編集してください。
- ビンゴカードは公式配列ルールに沿い、列ごとに以下の範囲から重複なしでランダム採用: B=1-15, I=16-30, N=31-45（中央FREE固定）, G=46-60, O=61-75。
